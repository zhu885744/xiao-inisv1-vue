# xiao-inisv1-vue_v1.0.1 更新日志

## 发布日期
2026-05-05

## 概述
本次更新主要聚焦于性能优化、API调用优化、用户体验提升以及代码质量改进。

---

## 性能优化

### 维护页面优化
- 使用 `requestAnimationFrame` 替代 `setInterval` 进行倒计时更新，减少性能损耗
- 增强时间解析函数，支持秒级时间戳和日期字符串格式

### 缓存机制优化
- **全局**：实现多层次缓存策略，根据数据特性设置不同缓存过期时间
- **搜索页面**：搜索结果添加5分钟缓存，避免重复查询
- **标签页面**：标签文章计数改为批量并行请求（每批5个标签）
- **分类页面**：分类文章添加60分钟缓存
- **作者页面**：用户信息添加30分钟缓存
- **侧边栏**：等级数据添加1小时缓存
- **自定义代码**：配置数据添加24小时localStorage缓存

### 并行请求优化
- **归档页面**：文章操作和评论并行加载
- **分类页面**：分类详情和文章列表并行请求
- **首页**：统计数据和文章列表并行获取
- **友链页面**：页面数据和友链列表并行加载
- **留言页面**：评论和统计数据并行获取

---

## 用户体验

### 搜索页面优化
- 新增热门搜索关键词展示
- 支持搜索范围切换自动触发搜索
- 搜索结果关键词高亮显示
- 支持键盘上下键选择、回车键确认跳转
- 路由参数同步（支持分享搜索链接）
- 无结果时推荐热门搜索

### 页面过渡动画
- 优化路由切换过渡动画，支持 `prefers-reduced-motion` 媒体查询
- 添加 Suspense 加载状态，提升初始化体验

### 友链状态监测
- 友链页面新增在线状态指示（🟢在线/🔴掉线）
- 显示响应时间（ms）

### 维护页面用户体验
- 添加刷新页面按钮，支持点击和快捷键（F5/Ctrl+R）刷新
- 维护结束后自动跳转至首页，并有动画提示
- 添加社交链接展示（GitHub、Twitter、微博）

## UI改进

### Bootstrap 5 集成
- 主题升级页面使用 Bootstrap 5 原生组件
- 适配暗黑模式样式
- 搜索结果选中状态高亮

### 维护页面样式优化
- 倒计时方块添加悬停动画效果
- 添加社交链接图标样式
- 支持 `prefers-reduced-motion` 媒体查询，尊重用户动画偏好

### 懒加载修复
- 修复标签页面文章封面懒加载显示问题
- 修正 `querySelectorAll` 无效选择器错误

## Bug修复

### 路由守卫
- 修复 `/user` 页面登录状态校验问题，避免重定向到首页

### 组件问题
- 移除 user.vue 中未使用的变量（loading、isLogin、toast、cache、onUnmounted、watch）
- 修复 search.vue 重复定义 onMounted 的问题

## 可靠性提升

### WebSocket 重连机制
- 实现指数退避重连策略（最多5次）
- 每次重试间隔指数增长，避免频繁重试

### 错误处理
- 统一 API 请求错误处理
- 移除不必要的 console.error 输出

---

## 代码质量

### 组件懒加载
- App.vue 中 nav、sidebar、footer、float-buttons、upgrade-modal 等组件改为异步加载
- 减少首屏加载体积

### 代码结构优化
- 使用 computed 属性替代模板中的直接表达式
- 脚本延迟注入（100ms）避免阻塞首屏渲染
