# xiao-inisv1-vue_v1.0.3 更新日志

## 发布日期
2026-05-12

## 概述
本次更新主要优化调整了用户协议和隐私协议功能。

## 功能新增

### 协议弹窗功能
- `auth.vue` 新增协议弹窗组件，支持显示《用户协议》和《隐私协议》内容
- 点击协议链接时弹出模态框展示完整协议内容
- 支持纯文字显示，保留换行格式，提高可读性
- 协议内容支持在后台自定义配置

### 协议内容管理
- `functions.vue` 新增协议内容编辑表单
- 支持编辑用户协议内容（多行文本）
- 支持编辑隐私协议内容（多行文本）
- 配置自动保存到站点设置，即时生效

## 默认协议内容
- 用户协议：包含服务条款、账户安全、用户行为规范、知识产权、服务变更、免责声明、协议变更等章节
- 隐私协议：包含收集的信息、信息使用、信息共享、数据安全、用户权利、Cookie使用、政策变更等章节

## 影响模块
- `src/comps/index/dialog/auth.vue`
- `src/views/index/pages/functions.vue`


# xiao-inisv1-vue_v1.0.2 更新日志

## 发布日期
2026-05-07

## 概述
本次更新主要修复了组件错误、优化了 API 调用机制、增强了版本检测功能。

## Bug修复

### 组件错误修复
- 修复 footer.vue 中 `fetchSystemVersion` 未定义错误
- 移除不再使用的 systemVersion 相关代码
- 移除不必要的 console.log 输出

### API 调用优化
- 修复 App.vue WebSocket 重复重连问题（移除与 socket.js 冲突的重连代码）
- 优化 socket.js 重连日志，减少频繁日志输出

## 性能优化

### API 调用去重
- `comm.js` 的 `fetchSiteInfo` 添加 Promise 缓存机制
- 防止多个组件同时调用时产生重复 API 请求
- 多个组件同时调用只会发起一个实际的 API 请求
- 对独立页面page.vue的 API 请求进行了全面优化

## 功能增强

### 版本检测功能
- `theme-upgrade.vue` 使用 `i-markdown.vue` 组件渲染 Markdown
- 支持代码高亮、复制功能，样式与文章页一致
- 版本历史使用 Bootstrap 5 模态框展示
- 支持滚动查看所有历史版本

## 影响模块
- `src/views/index/pages/page.vue`

# xiao-inisv1-vue_v1.0.1 更新日志

## 发布日期
2026-05-05

## 概述
本次更新主要聚焦于性能优化、API调用优化、用户体验提升以及代码质量改进。

## 性能优化

### 维护页面优化
- 使用 `requestAnimationFrame` 替代 `setInterval` 进行倒计时更新，减少性能损耗
- 增强时间解析函数，支持秒级时间戳和日期字符串格式

### 缓存机制优化
- **全局**：实现多层次缓存策略，根据数据特性设置不同缓存过期时间
- **搜索页面**：搜索结果添加5分钟缓存，避免重复查询
- **标签页面**：标签文章计数改为批量并行请求（每批5个标签）
- **分类页面**：分类文章添加60分钟缓存
- **作者页面**：用户信息添加30分钟缓存
- **侧边栏**：等级数据添加1小时缓存
- **自定义代码**：配置数据添加24小时localStorage缓存

### 并行请求优化
- **归档页面**：文章操作和评论并行加载
- **分类页面**：分类详情和文章列表并行请求
- **首页**：统计数据和文章列表并行获取
- **友链页面**：页面数据和友链列表并行加载
- **留言页面**：评论和统计数据并行获取

## 用户体验

### 搜索页面优化
- 新增热门搜索关键词展示
- 支持搜索范围切换自动触发搜索
- 搜索结果关键词高亮显示
- 支持键盘上下键选择、回车键确认跳转
- 路由参数同步（支持分享搜索链接）
- 无结果时推荐热门搜索

### 页面过渡动画
- 优化路由切换过渡动画，支持 `prefers-reduced-motion` 媒体查询
- 添加 Suspense 加载状态，提升初始化体验

### 友链状态监测
- 友链页面新增在线状态指示（🟢在线/🔴掉线）
- 显示响应时间（ms）

### 维护页面用户体验
- 添加刷新页面按钮，支持点击和快捷键（F5/Ctrl+R）刷新
- 维护结束后自动跳转至首页，并有动画提示
- 添加社交链接展示（GitHub、Twitter、微博）

## UI改进

### Bootstrap 5 集成
- 主题升级页面使用 Bootstrap 5 原生组件
- 适配暗黑模式样式
- 搜索结果选中状态高亮

### 维护页面样式优化
- 倒计时方块添加悬停动画效果
- 添加社交链接图标样式
- 支持 `prefers-reduced-motion` 媒体查询，尊重用户动画偏好

### 懒加载修复
- 修复标签页面文章封面懒加载显示问题
- 修正 `querySelectorAll` 无效选择器错误

## Bug修复

### 路由守卫
- 修复 `/user` 页面登录状态校验问题，避免重定向到首页

### 组件问题
- 移除 user.vue 中未使用的变量（loading、isLogin、toast、cache、onUnmounted、watch）
- 修复 search.vue 重复定义 onMounted 的问题

## 可靠性提升

### WebSocket 重连机制
- 实现指数退避重连策略（最多5次）
- 每次重试间隔指数增长，避免频繁重试

### 错误处理
- 统一 API 请求错误处理
- 移除不必要的 console.error 输出

## 代码质量

### 组件懒加载
- App.vue 中 nav、sidebar、footer、float-buttons、upgrade-modal 等组件改为异步加载
- 减少首屏加载体积

### 代码结构优化
- 使用 computed 属性替代模板中的直接表达式
- 脚本延迟注入（100ms）避免阻塞首屏渲染

## 版本检测

### GitHub Releases 集成
- 主题升级页面使用 GitHub Releases API 获取版本信息
- 支持版本更新检查、更新日志展示
