## 功能新增

### 协议弹窗功能

### 协议内容管理
- `functions.vue` 新增协议内容编辑表单
- 支持编辑用户协议内容（多行文本）
- 支持编辑隐私协议内容（多行文本）
- 配置自动保存到站点设置，即时生效

## 默认协议内容
- 用户协议：包含服务条款、账户安全、用户行为规范、知识产权、服务变更、免责声明、协议变更等章节
- 隐私协议：包含收集的信息、信息使用、信息共享、数据安全、用户权利、Cookie使用、政策变更等章节

## 性能优化

### 首屏加载优化
- **Vite 依赖预构建**：扩大 `optimizeDeps.include` 范围，将 vue-router、pinia、axios、bootstrap 等常用库纳入预构建，大幅减少首次构建时间
- **代码分包策略**：优化 `manualChunks` 分包策略，将大型依赖（highlight.js 963KB、markdown编辑器 873KB）单独打包，支持浏览器并行加载
- **CSS 按需分割**：启用 `cssCodeSplit: 'divider'` 按需分割 CSS，减少首屏 CSS 体积
- **并行初始化**：将站点信息获取与路由准备从串行改为并行执行，减少首屏等待时间

### 缓存策略优化
- **站点信息缓存**：`fetchSiteInfo` 添加 30 分钟缓存，避免每次刷新重复请求 API
- **首页文章列表**：有缓存时立即显示，后台静默刷新，用户体验显著提升
- **显示模式读取**：优先从 store 的 siteInfo 缓存读取 display_mode，不再重复请求 API
- **友链缓存时间**：page.vue 友链缓存时间从 60 分钟调整为 1440 分钟（1天）

### 懒加载优化
- **标签页面**：修复 keep-alive 缓存后文章封面图不显示的问题
- **图片加载检测**：添加 `img.complete && img.naturalWidth > 0` 检测，处理 Vue DOM 复用场景
- **观察标记清除**：缓存数据时正确清除 `data-observed` 和 `data-loaded` 标记
