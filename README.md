## inis **buyu** 单栏主题
基于 Vue 3 + js + Vite 开发的 inis 主题

## 功能特点
完全的响应式设计
极高速度的

## 开发
```
# 安装项目依赖
npm install

# 开启本地开发服务器（默认地址：http://localhost:5173，热更新实时预览）
npm run dev
```

## 打包
```
# 构建生产环境包（打包后生成dist目录，包含压缩后的静态文件，可直接部署）
npm run build

# 本地预览生产打包结果（验证打包后功能是否正常，无需部署服务器）
npm run preview
```
