## inis **buyu** 单栏主题
基于 Vue 3 + js + Vite 开发的 inis 主题

## 功能特点
完全的响应式设计
极高速度的

## 开发
```
# 安装项目依赖
npm install

# 开启本地开发服务器（默认地址：http://localhost:3000，热更新实时预览）
npm run dev
```

## 打包
```
# 构建生产环境包（打包后生成dist目录，包含压缩后的静态文件，可直接部署）
npm run build

# 本地预览生产打包结果（验证打包后功能是否正常，无需部署服务器）
npm run preview
```

## bug记录
1.友链数据无法正常获取，有重复
2.首页公告弹窗未适配暗黑模式

小型项目：保留原有逻辑，仅需增加心跳机制（解决连接假死）；
中大型项目：改为类模式 + 私有回调 + 消息队列，支持多实例、高并发；
对稳定性要求极高的场景：增加心跳 + 重连回调 + 断线缓存，提升用户体验。
