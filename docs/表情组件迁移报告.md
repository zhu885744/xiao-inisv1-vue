# 表情组件迁移报告

## 迁移概述

本次迁移将 `i-comment.vue` 和 `page.vue` 中的内联表情面板代码替换为统一的 `i-emoji-picker.vue` 组件，实现了表情功能的模块化和复用。

## 迁移详情

### 1. i-comment.vue 迁移

#### 修改内容：

**导入部分：**
- 移除了 `import OwOData from '@/assets/json/OwO.json'`
- 新增 `import iEmojiPicker from './i-emoji-picker.vue'`

**模板部分：**
- 移除了内联的表情面板模板代码（约 30 行）
- 使用 `<i-emoji-picker>` 组件替换，支持 v-model 双向绑定

**脚本部分：**
- 移除了以下不再需要的 ref：
  - `owoEmojis`
  - `activeEmojiCategory`
  - `toggleEmojiPicker` 函数
  - `toggleReplyEmojiPicker` 函数
  - `insertEmoji` 函数（保留，由组件事件调用）
  - `insertReplyEmoji` 函数（保留，由组件事件调用）
  - `handleClickOutside` 函数（已内置在组件中）

**样式部分：**
- 移除了约 90 行的表情相关 CSS 代码

#### 代码减少量：
- 模板：约 30 行 → 3 行
- 脚本：约 20 行
- 样式：约 90 行
- **总计减少约 140 行代码**

### 2. page.vue 迁移

#### 修改内容：

**导入部分：**
- 移除了 `import OwOData from '@/assets/json/OwO.json'`
- 新增 `import iEmojiPicker from '@/comps/custom/i-emoji-picker.vue'`

**模板部分：**
- 移除了两处内联的表情面板模板代码（留言表情面板和回复表情面板）
- 使用 `<i-emoji-picker>` 组件替换

**脚本部分：**
- 移除了以下不再需要的 ref：
  - `owoEmojis`
  - `activeEmojiCategory`
  - `toggleMessageEmojiPicker` 函数
  - `toggleReplyEmojiPicker` 函数
  - `handleClickOutside` 函数

**样式部分：**
- 移除了约 120 行的表情相关 CSS 代码

#### 代码减少量：
- 模板：约 60 行 → 6 行
- 脚本：约 20 行
- 样式：约 120 行
- **总计减少约 200 行代码**

## 组件使用方式

### 在 i-comment.vue 中：

```vue
<!-- 替换前：内联表情面板 -->
<div v-if="showEmojiPicker" class="emoji-picker-panel ...">
  <!-- 表情面板内容 -->
</div>

<!-- 替换后：使用通用组件 -->
<i-emoji-picker 
  v-model="showEmojiPicker"
  :is-dark-mode="isDarkMode"
  @select="insertEmoji"
/>
```

### 在 page.vue 中：

```vue
<!-- 替换前：内联表情面板 -->
<div v-if="showMessageEmojiPicker" class="emoji-picker-panel ...">
  <!-- 表情面板内容 -->
</div>

<!-- 替换后：使用通用组件 -->
<i-emoji-picker 
  v-model="showMessageEmojiPicker"
  :is-dark-mode="isDarkMode"
  @select="insertMessageEmoji"
/>
```
