# 全局代码功能优化重构说明文档

## 版本信息

## 优化内容概览

### 1. 组件优化

#### 新增通用表情组件 (`i-emoji-picker.vue`)

**功能特点**:
- 表情分类展示（颜文字、Emoji）
- 点击外部自动关闭
- 深色模式适配
- 响应式设计，支持移动端
- 动画过渡效果

**使用示例**:

```vue
<template>
  <div>
    <button @click="showPicker = true">选择表情</button>
    <i-emoji-picker 
      v-model="showPicker"
      :is-dark-mode="isDark"
      @select="onEmojiSelect"
    />
  </div>
</template>

<script setup>
import { ref } from 'vue'
import iEmojiPicker from '@/comps/custom/i-emoji-picker.vue'

const showPicker = ref(false)
const isDark = ref(false)

const onEmojiSelect = (emoji) => {
  console.log('选中的表情:', emoji)
}
</script>
```

### 2. 组合式函数 (Composables)

#### `useEmoji.js`

**功能**:
- 管理表情选择器状态
- 提供表情插入、追加功能
- 支持单个/多个表情选择器管理

**使用示例**:

```javascript
import { useEmoji, useMultipleEmoji } from '@/composables/useEmoji'

// 单个表情选择器
const { showEmojiPicker, toggleEmojiPicker, insertEmoji, appendEmoji } = useEmoji()

// 多个表情选择器（如主/从选择器）
const { primaryEmoji, secondaryEmoji, togglePrimary, toggleSecondary, closeAll } = useMultipleEmoji()
```

#### `useClickOutside.js`

**功能**:
- 检测元素外部点击
- 支持单个/多个元素检测
- 自动清理事件监听

**使用示例**:

```javascript
import { ref } from 'vue'
import { useClickOutside, useClickOutsideMultiple } from '@/composables/useClickOutside'

const el = ref(null)
useClickOutside(el, () => {
  console.log('点击了外部')
})

// 多个元素
const el1 = ref(null)
const el2 = ref(null)
useClickOutsideMultiple([el1, el2], () => {
  console.log('点击了所有元素的外部')
})
```

### 3. 工具函数 (Utils)

#### 验证工具 (`validators.js`)

**函数列表**:
- `validateComment()` - 评论内容验证
- `validateUsername()` - 用户名验证
- `validatePassword()` - 密码验证
- `validateEmail()` - 邮箱验证
- `checkRateLimit()` - 频率限制检查

**使用示例**:

```javascript
import { validateComment, checkRateLimit } from '@/utils/validators'

const result = validateComment('评论内容', { minLength: 1, maxLength: 1000 })
if (!result.valid) {
  console.error(result.message)
}

const rateCheck = checkRateLimit(lastCommentTime, 30)
if (!rateCheck.allowed) {
  console.warn(`请等待 ${rateCheck.remaining} 秒`)
}
```

#### 格式化工具 (`formatters.js`)

**函数列表**:
- `formatDate()` - 日期格式化
- `formatRelativeTime()` - 相对时间显示
- `formatNumber()` - 数字格式化（K/M）
- `truncateText()` - 文本截断
- `escapeHtml()` - HTML转义
- `stripHtml()` - 去除HTML标签

**使用示例**:

```javascript
import { formatRelativeTime, formatNumber } from '@/utils/formatters'

console.log(formatRelativeTime(new Date())) // '刚刚'
console.log(formatNumber(1500)) // '1.5K'
console.log(formatNumber(1500000)) // '1.5M'
```

### 4. 常量配置 (`constants/index.js`)

统一管理项目常量，包括:

```javascript
import { STORAGE_KEYS, API_ENDPOINTS, VALIDATION_RULES } from '@/constants'
```

### 5. 初始化优化 (`utils/init.js`)

将 `main.js` 中的初始化逻辑拆分到独立模块，提高可维护性。
