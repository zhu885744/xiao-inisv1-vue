## 应用概述

## 应用详情

### 1. validators.js - 验证工具

#### 应用位置
- **i-comment.vue** - 评论组件

#### 应用内容
- `validateComment()` - 评论内容验证
- `checkRateLimit()` - 频率限制检查

#### 代码修改
```javascript
// i-comment.vue

// 导入
import { validateComment, checkRateLimit as checkRateLimitUtil } from '@/utils/validators'

// 使用 validateComment
const validateCommentContent = (content) => {
  const result = validateComment(content, {
    minLength: 1,
    maxLength: maxCommentLength.value
  })
  
  if (!result.valid) {
    Toast.error(result.message)
    return false
  }
  // ... 其他验证逻辑
}

// 使用 checkRateLimit
const checkRateLimit = () => {
  const rateLimit = commentConfig.value.rate_limit || {}
  if (rateLimit.enabled !== 1) return true
  
  const result = checkRateLimitUtil(lastCommentTime.value * 1000, rateLimit.time_window || 60)
  
  if (!result.allowed) {
    Toast.error(`评论过于频繁，请等待 ${result.remaining} 秒后再试`)
    return false
  }
  return true
}
```

### 2. formatters.js - 格式化工具

#### 应用位置
- **i-comment.vue** - 评论组件
- **page.vue** - 留言页面

#### 应用内容
- `formatDate()` - 日期格式化（支持 Unix 时间戳秒级自动转换）
- `formatRelativeTime()` - 相对时间显示

#### 代码修改

**i-comment.vue:**
```javascript
// 导入
import { formatDate } from '@/utils/formatters'

// 使用 formatDate
const formatTime = (timestamp) => {
  if (!timestamp || timestamp === 0) return '未知时间'
  return formatDate(timestamp, 'YYYY-MM-DD HH:mm')
}
```

**page.vue:**
```javascript
// 导入
import { formatDate } from '@/utils/formatters'

// 使用 formatDate
const formatTime = (timestamp) => {
  if (!timestamp || timestamp === 0) return '未知时间'
  return formatDate(timestamp, 'YYYY-MM-DD')
}
```

### 3. formatters.js 增强

#### 改进内容
- `formatDate()` 现在支持自动检测 Unix 时间戳（秒）
- 如果传入的时间戳小于 `1e12`（即小于 2001-09-09），自动乘以 1000 转换为毫秒
- 这使得函数可以兼容 `utils.timeToDate()` 的接口

#### 兼容性
```javascript
// 两种调用方式都可以
formatDate(Date.now())                    // 毫秒时间戳
formatDate(1717132800)                    // Unix 时间戳（秒）
formatDate(new Date())                   // Date 对象
formatDate('2024-05-31T12:00:00Z')       // ISO 字符串
```
